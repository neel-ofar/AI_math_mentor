# Coordinate Geometry

### Distance Formula
- Distance between (x1,y1) and (x2,y2): √[(x2-x1)² + (y2-y1)²]

### Section Formula
- Internal division: ( (mx2+nx1)/(m+n), (my2+ny1)/(m+n) )
- External division: ( (mx2-nx1)/(m-n), (my2-ny1)/(m-n) )

### Straight Lines
- Slope m = (y2-y1)/(x2-x1)
- Equation: y - y1 = m(x - x1)
- Slope-intercept: y = mx + c
- Intercept form: x/a + y/b = 1
- General form: Ax + By + C = 0

### Circle
- Standard: (x-h)² + (y-k)² = r²
- General: x² + y² + 2gx + 2fy + c = 0
- Center: (-g, -f), Radius: √(g² + f² - c)

### Conic Sections
- Parabola: y² = 4ax
- Ellipse: x²/a² + y²/b² = 1
- Hyperbola: x²/a² - y²/b² = 1
